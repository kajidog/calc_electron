# calc_electron

デスクトップアプリの電卓

## 使用言語

node v18.16.0  
react  
electron  
typescript  

## インストーラー

Windows/MacOS対応  
https://github.com/kajidog/calc_electron/releases/tag/0.0.0


## 開発方法

```bash
git clone https://github.com/kajidog/calc_electron.git
cd calc_electron
npm i
npm run dev
```
